# Insurance Operations Frontend

NextJS frontend for the Insurance Operations Platform.

## Local Run

```bash
cd frontend
npm install
npm run dev -- --port 3002
```

Open `http://localhost:3002`.

Current pages:

- `/`
- `/dashboard`
- `/insurance`
- `/ai`
- `/admin`

Set the backend URL with:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8002/api/v1
```
